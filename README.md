# PaperWM-Hammerspoon

在MacOS上实现了类似于Niri式的纸带式窗口平铺管理器的效果。

基于自行安装Hammerspoon，加上 https://github.com/mogenson/PaperWM.spoon 的扩展。

并在原版的基础上，修改了自己的快捷键，增加了两个功能

使用方法

自行安装Hammerspoon

RAW下载 https://github.com/mogenson/PaperWM.spoon 仓库。解开Zip，将得到的文件夹重命名为：PaperWM.Spoon，
双击之，会自动导入Hammerspoon。

点击Hammerspoon，Open Config
